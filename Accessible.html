Combat Robotics on a budget

Cost is often the biggest barrier to entry for people who want to try out the sport of combat robotics. As an experiment I tried to build these robots as cheaply as possible.

Hades is a 3lb combat robot that I built in 2 days and with access to few tools.

This robot was built for under 50 dollars with mostly Aliexpress parts.

This robot was a test for an event where people would build combat robots with foam core and hot glue.
